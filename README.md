**Clean flash**

• Use #Recovery given in notes  

CLEAN FLASH ( Gapps Variant )
1. Boot into Recovery 
2. Wipe dalvik and cache 
3. Flash firmware ( skip if already on hyper os fw )
4. Flash Rom ( tick reflash recovery option )
5. Format data 
6. Reboot system 

CLEAN FLASH  ( Vanilla Variant )
1. Boot into Recovery 
2. Wipe dalvik and cache 
3. Flash Firmware ( skip if already on hyper os fw )
4. Flash Rom ( tick reflash recovery option )
5. Reboot Recovery 
6. Flash Gapps
7. Format data
8. Reboot system
